# 🌐 Personal Portfolio Website

A simple and responsive portfolio website built with **HTML, CSS, and JavaScript**.  
This project showcases my skills, projects, and services, with a clean design and easy navigation.

---

## ✨ Features
- **Home Section** – Welcoming introduction with a clear headline and navigation.
- **About Me Section** – Highlights background, skills, and passion for web development.
- **My Services Section** – Outlines key offerings such as web development, responsive design, and UI/UX improvements.
- **Contact Me Section** – Includes a functional contact form integrated with EmailJS for direct communication.

---

## 🛠️ Tech Stack
- **HTML5** – Structure and content
- **CSS3** – Styling and responsive layout
- **JavaScript** – Interactivity and dynamic features
- **EmailJS** – Contact form integration
